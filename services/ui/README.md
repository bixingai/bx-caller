# BX Caller Web Console (UI scaffold)

This is a dedicated web-console scaffold for call-center operations.

## Included product modules
- Login/API token connect
- Agent view
- Calls view
- Transcript/QA placeholder
- Analytics placeholder
- Admin settings placeholder

## Run locally
Serve the files with any static server from this directory, for example:

```bash
cd /home/runner/work/bx-caller/bx-caller/services/ui
python -m http.server 3000
```
